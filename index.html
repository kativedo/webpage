GraphQL

GraphQL was released by Facebook in 2015 and is gaining popularity. It's a data query language and is a better alternative to REST API and ad-hoc webservice architectures. It allows the client to specify exactly what data it needs, makes it easier to collect data from multiple sources, and it uses a type system to describe data.
